Contact the vim-latex-devel team

vim-latex-devel[AT]lists.sourceforge.net is a list meant for discussing bugs/feature request and other things pertaining to latex-suite. It is not meant to be a developers only list as the name might imply. It is a closed list. This means that you can only e-mail vim-latex-devel if you are subscribed to it. If you wish to subscribe, go to this page. vim-latex-devel is moderately active list which gets about 3-4 mails per day on average.
